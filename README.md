# Tweeter

A bot to use twitter from the command line on the fly!

## Demo

![](assets/demo.gif)

## Commands

1. Get all commands 
```bash
$ tweeter -h
```

2. Set Twitter keys
```bash
$ tweeter keys set
```

3. Tweet something
```bash
$ tweeter tweet message
```

4. Get tweets
```bash
$ tweeter show tweets
```

5. Seach for tweets based on a keyword
```bash
$ tweeter search tweets
```
